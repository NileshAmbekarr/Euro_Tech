## 🐛 Troubleshooting

### If Login/Register Doesn't Work:
1. Check backend server is running
2. Verify API endpoints in environment files
3. Check browser console for errors
4. Ensure CORS is configured on backend

### If Contact Form Doesn't Submit:
1. Verify backend contact API is running
2. Check network tab for API call
3. Ensure form validation passes
4. Check backend logs for errors

### If Styling Looks Wrong:
1. Clear browser cache
2. Hard refresh (Ctrl+Shift+R)
3. Check CSS files loaded in Network tab
4. Verify no CSS conflicts

### If Images Don't Load:
1. Check file paths are correct
2. Verify assets folder exists
3. Check image file names match code
4. Look for 404 errors in console
